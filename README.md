
# Freewheel Rentals


A small project where people can sell and rent their vehicle is very useful for employees and students.


## Deployment

To deploy this project for front-end

```bash
  cd client
```
```bash
  npm i
```
```bash
  npm run start
```

To deploy this project for back-end

```bash
  cd server
```
```bash
  npm i
```
```bash
  node server.js
```
